Telescope

GUI for Telescope, an open-source, peer-to-peer, and decentralized file storage system based on IPFS

This project was founded by Josselin Chevalay to make IPFS more user-friendly with a graphic user interface (GUI). I redesigned the UX/UI to improve the usability and aesthetic of the program.

Concept

We decided to base the UI on popular cloud-storage systems that users are already familiar with, but with a more futuristic interface and IPFS functionalities.

Sketches

Wireframe

We wanted to showcase the immutable and peer-to-peer nature of IPFS, so we implemented a progress bar that would continually show the hash generation progress and node connection status.
