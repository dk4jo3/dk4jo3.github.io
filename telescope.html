Telescope

An Open-source Decentralized File Storage GUI for IPFS

This project was started by Josselin with the goal of making IPFS more user-friendly with a more user-centered interface. I created UX/UI design and developed partial front-end with HTML/CSS.

Concept

The idea is to build a user-centric file management system with back-end connected to ipfs network. We wanted it to be based on other popular storage system that users are already familiar with but with a more futuristic interface and added functionalities.

Sketches

Wireframe

We wanted to create a familiar experience for the users with some elements that represent the immutable, peer-to-peer nature of ipfs so we implemented a progress bar on top that would constatnly run, showing hash generation or node connection.
