Open Tax Solver

Open Tax Solver is an open-source tax filing software that offers users a free, and secure tax filing solution. I redesigned the UX/UI of the program to improve its usability, appearance, and features.

UI Issues

The user interface, landing page, and installation process are outdated and lacks usability. Moreover, users are unable to file electronically via OTS. They would have to print and mail the forms. Yet, OTS fails to provide any assistance to assist its users in that matter.

Target Users and Journey Maps

Based on the insights from target users analysis and journey maps, I identified multiple pain points throughout their experience and made design suggestions for improving the experience.

    Design Suggestions
  • Updated landing page and installation process
  • Tools to help users complete the forms
  • Tools to help users select the forms they need
  • Explicit mailing instructions and tracking information.

Sketches

Main Page

The main page showcases the three primary steps of filing taxes with Open Tax Solver. Users can start a new filing or load the previous filing on this page.

Questionairre

A short questionnaire can help users determine the required tax forms for them. For more advanced users, they can skip the questionnaire to select the forms directly.

Helper Dialogue

An interactive dialogue can help users with difficult parts of the forms. In this example, users can find out the dependent qualification by answering the questions in the dialogue.

Tally Tool

A tally dialogue can help users tally the number inside the program. It also records the entered line items for future review.

Report

As Open Tax Solver does not support the e-filing service, it displays users’ respective tax agencies’ addresses. Users can save or print the finished tax forms.
