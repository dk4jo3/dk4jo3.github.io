HABEO

How might we help homeowners understand their utility consumption on a deeper level?

This project explored the possibilities of using Internet of Things device to help homeowners manage utility usage and be more aware of their utility usage behavior.

I contributed to the design of the animations of the application, gestural controls, user interface, and conducting interviews for this project.

Target Users

This project first began by selecting the target user groups of the application and creating personas to more effectively envision the different behaviors and motivations the target user group might have.

We focused on the users who would benefit from this application the most; landlords and new property owners. Landlords have multiple properties to manage and new homeowners are often unfamiliar with the ins and outs of owning a home. We then developed an experience map based on the personas to further understand their behavior, wants, and needs.

Interviews

We developed the main features by envisioning potential problems the target users might encounter, then brainstorm potential solutions for these issues. After deliberation, we determined the main features of Habeo to be the ability to visualize real-time data, compare utility consumptions from the past, manage usage goals, and detect abnormal usage behavior.

Paper Prototype

We developed the low-fidelity UX through sketching and creating rapid prototypes. We created a rough wireframe of the application and established a basic style of data visualization. With the information, we created a series of paper prototypes for usability testing.

Usability Testing

We gave participants varied sets of tasks in order to understand how they use and perceive the features. Often, users would ask what a certain button does but instead of answering their question, we would ask them what they think would happen if they pressed the button. We also asked them about what they think was successful, what was not, and what can be improved.

Gesture – We modified the gesture control so it would correspond with the animation in the application, We also added hints to assist with the navigation.

Time – We redesigned and simplified the comparison feature. Users can use consistent the gestural controls to quickly select time frames to compare, instead of inputting specific dates numerically.

Home – We removed the logo on the top right corner as some users confused it with a home button.

What is Habeo?

Habeo is a solution to the problem of homeowners not understanding what their utility consumption behaviors. Habeo creates a transparent and more direct dialogue between the users and their utility consumption.

The data flow of Habeo starts from the sensors that are installed on the meters of the respective utilities. The sensors record the information and relay the data to the property’s wireless router via WI-Fi. The system then sends the data to a cloud server that is accessible from the application.
